## O Que é C?

C é uma linguagem de programação de propósito geral criada por Dennis Ritchie em 1972 nos Bell Labs. É uma das linguagens mais utilizadas de todos os tempos e serve como base para muitas linguagens modernas como C++, Java e Python.

## Por Que Aprender C?

- **Performance** — C oferece controle detalhado sobre hardware e memória, sendo ideal para programação de sistemas.
- **Base sólida** — Entender C ajuda você a compreender como os computadores realmente funcionam em baixo nível.
- **Portabilidade** — Programas em C podem ser compilados em praticamente qualquer plataforma.
- **Uso na indústria** — Sistemas operacionais (Linux, Windows), sistemas embarcados, bancos de dados e engines de jogos são feitos em C.
- **Adaptabilidade** — Muitas linguagens de programação são parecidas com ou baseadas em C, isso te poupará bastante tempo no futuro.
